# User
Creates the users and groups required by the company.

## Tasks

### Create Developer Group
This task creates the developer group.

### Create Admin Group
This task creates the admin group.

### Give Admin Sudo
This task gives sudo access to the admin group.

### Add the User 'Hermione'
This task adds a user named hermione and adds them to the developer group.

### Add the User 'Katniss'
This task adds a user named katniss and adds them to the developer group.

### Add the User 'Ian'
This task adds a user named ian and adds them to the admin group.
